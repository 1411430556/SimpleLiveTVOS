## 背景：

遇到一个非常好的项目:  [dart_simple_live](https://github.com/xiaoyaocz/dart_simple_live/) `原项目`

原项目基于Flutter制作，不支持tvOS端，于是作为一个原项目的补充。

## 支持平台：

tvOS 17+ 

## 功能：

* [x] 观看直播

* [x] 切换线路/清晰度

* [x] 弹幕

* [x] 收藏（iCloud）

* [x] 支持关键词/房间号/平台长短链接/分享码 搜索直播间
